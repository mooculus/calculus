\begin{exercise}
Calculate $\int 4x \sin\left(x^2\right) \d x $.


\[
 \int 4x \sin\left(x^2\right) \d x = \answer{ -2\cos\left(x^2\right)  + C } 
\]
(Use $C$ for the constant of integration)

\end{exercise}
