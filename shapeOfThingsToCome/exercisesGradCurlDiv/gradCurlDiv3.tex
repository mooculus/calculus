In the exercises that follow, consider the function $\vec{F} : \R^3 \to \R^3$ defined by the rule $\vec{F}(x,y,z) = \vector{z,x + y + z,y + 2 z}$.

\begin{exercise}
  Compute \(\divergence \vec{F}\).
  \begin{multipleChoice}
    \choice{This is zero.}
    \choice[correct]{This can be computed (and is nonzero).}
    \choice{That does not even make sense.}
  \end{multipleChoice}
  \begin{exercise}
    \[
      \divergence \vec{F} = \answer{ 3 }
    \]
  \end{exercise}
\end{exercise}
