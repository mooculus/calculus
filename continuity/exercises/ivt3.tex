\begin{exercise}
The Intermediate Value Theorem states:
If $f$ is a continuous function for all $x$ in the closed interval
$[n,b]$ and $d$ is between
\wordChoice{\choice{$n$}\choice[correct]{$f(n)$}} and
\wordChoice{\choice{$b$}\choice[correct]{$f(b)$}}, then there is a
number \wordChoice{\choice[correct]{$c$}\choice{$f(c)$}} in
  \wordChoice{\choice{$[f(n),f(b)]$}\choice[correct]{$[n, b]$}} such that
  \wordChoice{\choice[correct]{$f(c) = d$}\choice{$f(d) = c$}}.
