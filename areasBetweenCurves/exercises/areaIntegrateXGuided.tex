\begin{exercise}
The following is a guided exercise to find the area of the region bounded $y = -x^2+4x+3$ and $y =-x^3+7x^2-10x+5$ between $x = 1$ and $x=2$.

\begin{exercise}
In order to express this area as a single integral, we should:

\begin{multipleChoice}
\choice[correct]{integrate with respect to $x$.}
\choice{integrate with respect to $y$.}
\end{multipleChoice}


\begin{exercise}
Since we are computing the area as an integral with respect to $x$ , we must use 
\begin{multipleChoice}
\choice[correct]{vertical slices.}
\choice{horizontal slices.}
\end{multipleChoice}



\begin{exercise}

Since we want to integrate with respect to $x$, we need to express the curves as functions of $x$, the limits of integration must be $x$ limits, and we need to find $h$ in terms of $x$.  The curves are already written in terms of $x$, so we need to worry about finding the limits and $h$ only.

The lower limit of integration is $x= \begin{prompt} \answer{1} \end{prompt}$ 

The upper limit of integration is $x= \begin{prompt} \answer{2} \end{prompt}$


We must now express $h$ in terms of the variable of integration.  Since $h$ is a vertical distance:
\begin{multipleChoice}
\choice[correct]{$h= y_{top}-y_{bot}$}
\choice{$h = x_{right}-x_{left}$}
\end{multipleChoice}


\begin{exercise}
The function used to determine the upper $y$-value, $y_{top}$ is:
\begin{multipleChoice}
\choice[correct]{$y_{top}=-x^2+4x+3$}
\choice{$y_{top}=-x^3+7x^2-10x+5$}
\end{multipleChoice}

The function used to determine the lower $y$-value, $y_{bot}$ is:
\begin{multipleChoice}
\choice{$y_{bot}=-x^2+4x+3$}
\choice[correct]{$y_{bot}=-x^3+7x^2-10x+5$}
\end{multipleChoice}

The height $h$ of the rectangle is thus $h=y_{top}-y_{bot} = \answer{( -x^2+4x+3)-(-x^3+7x^2-10x+5)}$.

\begin{exercise}
Thus, the area is given by:
  \[
 \int_{x=a}^{x=b} h \d x =  \int_{x=\answer{1}}^{x=\answer{2}} \answer{( -x^2+4x+3)-(-x^3+7x^2-10x+5) } \d x = \answer{\frac{49}{12}}.
  \]

\end{exercise}  
\end{exercise}
\end{exercise}
\end{exercise}
\end{exercise}
