\begin{exercise}

Consider the function $f(x) = e^{2x}$.

The linear approximation of $f$ at $a=0$ is 
\[
L(x) = \answer{1+2x }.
\]

Using this linear approximation, approximate the value of $e$.

\begin{hint}
Note that $e=e^1=e^{2(\frac{1}{2})}=f\left(\frac{1}{2}\right)$.
\end{hint}

\[
e \approx \answer{2}.
\]

\end{exercise}
